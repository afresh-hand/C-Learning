# Chapter 2
## 函数
* 定义： 函数头（返回类型和参数） + 函数体
* 在使用之前一定 要提供函数原型 （函数头 + ；）

## 命名空间
在使用的时候可以分为四种情况
1. 在函数内使用 using则函数内可以使用对应命名空间下的函数
2. 在函数定义之前，当前文件内所有函数都可以使用
3. 指定一个函数 如 using std::cout 则该函数被引用 其他的无法访问
4. 不使用using 在使用时制定命名空间 如  
`
std::cout << 'world' << std::endl
`
